# Панель прораба: ремонт квартиры

Учебный web-прототип панели прораба для показа заказчику хода ремонта квартиры по этапам.

## Что показывает

- общий процент готовности ремонта;
- сводку по этапам: завершено, в работе, ожидает;
- карточки этапов ремонта квартиры;
- фото-заглушки вместо настоящих файлов;
- комментарии прораба;
- пример замечания заказчика;
- панель замечаний со статусами: новое, в работе, закрыто;
- учебные состояния: заполнено, пусто, ошибка, успех.

## Что не делает

Проект специально не добавляет backend, базу данных, регистрацию, настоящую загрузку фото, чат, уведомления, оплату, API-ключи, персональные данные, Telegram и production SaaS.

## Как открыть

Самый простой способ: открыть `index.html` в браузере.

Если нужен локальный запуск, можно выполнить:

```powershell
powershell -ExecutionPolicy Bypass -File ./serve.ps1
```

После запуска открой адрес, который появится в окне терминала.

## Как проверить

Если на компьютере установлен Node.js:

```powershell
npm test
```

Проверка смотрит основные элементы страницы, учебные состояния и отсутствие типичных ошибок кодировки.

## Главные файлы

- `index.html` - структура панели прораба;
- `styles.css` - внешний вид и мобильная адаптация;
- `app.js` - переключение учебных состояний;
- `PROJECT_STATUS.md` - описание идеи и границ учебной версии.
